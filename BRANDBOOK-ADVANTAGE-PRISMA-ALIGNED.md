## üéì DESIGN RATIONALE: PRISMA COHERENCE

### Why This System Maintains Prisma Integrity

**1. Color Coherence (Black/White Base + Gold Extension)**
- **Prisma Standard:** Black/White + Cyan/Purple/Pink accents
- **ADVANTAGE Addition:** Gold as NEW sub-brand color (doesn't conflict with existing products)
- **Psychology:** Gold = opportunity, achievement, value (parent investment mindset)
- **Ecosystem Fit:** Gold sits between Prisma's warning Yellow and pure luxury, appropriate for youth pathway
- **Rationale:** Youth program needs warmth (vs corporate cold), Gold delivers without breaking Prisma restraint

**2. Typography Consistency (Inter + JetBrains Mono)**
- **Prisma Standard:** Inter for all content, JetBrains Mono for metrics/code
- **ADVANTAGE Application:** Same typefaces, same hierarchy, same credibility cues
- **Parent Trust:** Metrics in mono (85%, 40 cupos) signal precision and data-driven approach
- **Rationale:** Parents evaluate executive education quality; mono typography = professional credibility

**3. Visual Restraint (No Diagonal Chaos)**
- **Prisma Standard:** "Antidote to thumbnail aesthetics" ‚Äî clean grids, whitespace authority
- **ADVANTAGE Temptation:** Youth = energy = diagonals/chaos (REJECTED)
- **Implementation:** Strict grids, subtle Gold accents only, 40% whitespace maintained
- **Rationale:** Parents are the buyers (38-52 years old); they expect Prisma-level sophistication, not youth marketing

**4. Motion Discipline (200-300ms, Subtle)**
- **Prisma Standard:** Motion restraint (no loops on editorial, 2-4px hover lifts)
- **ADVANTAGE Application:** Same duration, same easing, same subtlety
- **Rationale:** Excessive motion = unprofessional; Prisma parents expect executive-grade polish

**5. Glassmorphism (5-10% Blur, 1px Borders)**
- **Prisma Standard:** Subtle depth without heavy effects (legibility priority)
- **ADVANTAGE Application:** Identical implementation across cards, overlays, navigation
- **Rationale:** Visual consistency across Prisma ecosystem (ADVANTAGE cards could appear in Prisma newsletter)

**6. Sub-Brand Flywheel Integration**
- **Prisma System:** Each product gets ONE icon + ONE color (Ojos/Cyan, Explosi√≥n/Pink, etc.)
- **ADVANTAGE Fit:** Trajectory/Gold extends system coherently (pre-funnel youth pathway)
- **Strategic Role:** Today's ADVANTAGE students = tomorrow's Cafes/Product Nights participants
- **Rationale:** 10-15 year relationship cultivation starts with youth program entry point
